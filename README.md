# 🎀 Coquette Card Maker

Website pembuat kartu ucapan dengan tema coquette yang cute dan estetik! Buat kartu ucapan terima kasih, ulang tahun, dan cinta dengan mudah dan download sebagai gambar PNG.

## ✨ Fitur Utama

- 🌸 **Opening Animation** - Sambutan hangat dengan animasi bunga berguguran
- 🎨 **3 Jenis Kartu**:
  - Kartu Terima Kasih (Thank You)
  - Kartu Ucapan Cinta (With Love)
  - Kartu Ulang Tahun (Happy Birthday)
- 🎨 **7 Pilihan Warna Pastel**:
  - Soft Mint (Hijau Muda)
  - Sage Green (Hijau Tua)
  - Teal Light (Biru Teal)
  - Coquette Pink (Pink Pastel)
  - Soft Yellow (Kuning Pastel)
  - Sky Blue (Biru Langit)
  - Soft Purple (Ungu Pastel)
- ✍️ **Customizable Text**:
  - Nama penerima (To)
  - Isi pesan
  - Nama pengirim (From)
- 📥 **Download as PNG** - Simpan kartu sebagai gambar berkualitas tinggi
- 📱 **Responsive Design** - Bekerja sempurna di desktop dan mobile
- 🎭 **Beautiful Typography** - Menggunakan Google Fonts dengan kaligrafi anggun

## 🚀 Cara Menggunakan

1. Buka website: [Coquette Card Maker](https://chikoosca.github.io/coquette-card)
2. Klik tombol "Enter Studio 🎀" pada opening screen
3. Pilih jenis ucapan yang ingin dibuat
4. Isi nama penerima, pesan, dan nama pengirim
5. Pilih warna latar yang disukai
6. Lihat preview kartu di sebelah kanan
7. Klik "Simpan Kartu (PNG)" untuk mengunduh

## 🛠️ Tech Stack

- **HTML5** - Struktur halaman
- **Tailwind CSS** - Styling modern
- **JavaScript** - Interaktivitas
- **Google Fonts** - Typography elegans
- **HTML2Canvas** - Screenshot to PNG
- **GitHub Pages** - Hosting gratis
